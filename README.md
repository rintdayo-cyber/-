## 🏃 紹介されている種目

1. **📣 応援** - クラスやチームを応援する種目
2. **⚪ 大玉転がし** - 大きなボールをみんなで協力して転がす
3. **🎈 大玉運び** - 大玉を落とさないように運ぶ
4. **🪆 だるま運び** - だるまを落とさないように走る
5. **🔄 リレー** - バトンを繋いで速さを競う
6. **🌪️ 台風の目** - 円陣を作って走る
7. **🔗 綱引き** - 全員で力を合わせて綱を引く

## 🚀 セットアップ

GitHub Pages を有効にするには、以下の手順を実行してください：

1. リポジトリの **Settings** に移動
2. 左側メニューで **Pages** を選択
3. **Source** を "Deploy from a branch" に設定
4. **Branch** を "main" に設定
5. **Folder** を "/ (root)" に設定
6. **Save** をクリック

約1～2分後、以下のURLでアクセスできるようになります：
```
https://rintdayo-cyber.github.io/-/
```
